# Ron Kommoji Portfolio

A brutalist minimalist portfolio website built with HTML, CSS, and Vanilla JavaScript.

## Design Philosophy
- **Aesthetic**: Editorial Minimalism with Technical Precision.
- **Key Features**:
  - Text-first content hierarchy.
  - Terminal status widget.
  - Interactive experience dropdowns.
  - Expanding bottom social navbar.

## Project Structure
```
portfolio/
├── index.html          # Main entry point
├── styles/
│   ├── main.css        # Global styles & variables
│   ├── components.css  # Component-specific styles
│   └── responsive.css  # Media queries
├── scripts/
│   └── main.js         # Interactivity
└── assets/
    ├── images/         # Profile pictures
    └── Ron_Kommoji_Resume.pdf
```

## How to Run Locally
1. Clone the repository.
2. Open `index.html` in your browser.
3. Or use a local server:
   ```bash
   npx serve portfolio
   ```

## Deployment
Ready to deploy on Vercel, Netlify, or GitHub Pages.
